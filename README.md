# Portafolio Personal - Pablo Salgado Hernandez

 Descripción
Este es mi portafolio web como parte de un proyecto para la asignatura Desarrollo Web Front-End en INACAP. Muestra mis habilidades en HTML5, CSS3 (Flexbox, Grid, Responsive) y JavaScript. Es una aplicación web interactiva y moderna.

## Características
- Single Page Application con navegación por anclas.
- Secciones: Hero (foto/tabla/bio), Habilidades (gráficos + soft-skills), Proyectos (galería Grid), Contacto (formulario validado), Footer.
- Responsive (funciona en móviles, tabletas y escritorios).
- Interactividad: Copiar teléfono, animaciones en barras de habilidades.
- Imágenes optimizadas para web.

## Tecnologías Usadas
- **Frontend**: HTML5, CSS3, Flexbox, Grid, Animaciones CSS.
- **Herramientas**: Git, GitHub.
- **Backend/Extras**: JavaScript (Vanilla).
